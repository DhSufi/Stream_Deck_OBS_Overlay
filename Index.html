Developers

The OBS Browser source will receive custom events that can be listened in JavaScript as follows: 

                
<!DOCTYPE html>

<script type="text/javascript">
    window.addEventListener('customEvent', function(event) {
    console.log(event.detail) // outputs: {"key": value, "key", "value", ...}
    })
</script>

Custom events

Event: slot_show
Content: {"slot": SLOT, "poke": "filename.png", "item": "filename.png"}

SLOT is String. Can be any of this values: "A1", "A2", "A3", "A4", "B1", "B2", "B3", "B4"

filename is String that finish with ".png". Can be any Pokémon ID/Item ID of the resource folder: "313_000.png", "128_002.png", ...


Event: slot_reset
Content: {"slot": SLOT}

SLOT is String. Can be any of this values: "A1", "A2", "A3", "A4", "B1", "B2", "B3", "B4"


Event: slot_hp
Content: {"slot": SLOT, "hp": FLOAT}

SLOT is String. Can be any of this values: "A1", "A2", "A3", "A4", "B1", "B2", "B3", "B4"

FLOAT from 0 to 1


Event: details
Content: {"nameA": "String", "recordA", "String", "countryA": "String", "nameB": "String", "recordB", "String", "countryB": "String"}

Be careful, any String can be "UNDEFINIED"


Event: controls
Content: {"data": DATA}

DATA is String. Can be any of this values: "show", "AP", "AS", "BP", "BS", "reset"
AP: player A score Add
AS: Player A score Substract
BP: player B score Add
BS: player B score Substract
reset: Reset scores
show: overlay visibility


Event: status
Content: {"slot": SLOT, "status": NAME}

SLOT is String. Can be any of this values: "A1", "A2", "A3", "A4", "B1", "B2", "B3", "B4"
NAME is String. Can be any of this values: "burn", "sleep", "paralysis", "freeze", "poison"


Event: faint
Content: {"slot": SLOT}

SLOT is String. Can be any of this values: "A1", "A2", "A3", "A4", "B1", "B2", "B3", "B4"
